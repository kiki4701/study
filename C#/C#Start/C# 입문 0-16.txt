C# 입문 0-16

보통 파일(여기서는 CH_16 클래스)이 추가되면 프로젝트(여기서는 FirstStudy) 우클릭해서 솔루션 다시빌드 를 해준다.

솔루션 다시빌드를 하게 소스파일과 소스들을 컴파일하고 연결해주는 역할을 한다.

class에서 선언한 변수 옆에 {get; set;}

이라고 입력해주면 다른곳에서 변수를 직접 초기화하고 가져다가 쓸 수 있다. (자바에서 getter / setter 느낌)

예를들면

Ch_16 staff1 = new Ch_16();

staff1.name = "핑크퐁";

Console.WriteLine(staff1.name);

이런식으로 초기화하고 불러서 쓸 수 있다.

하지만 이 경우 staff1 클래스의 name 변수가 public 이어야 한다.

그리고 만약 변수 옆에 {get;} 이렇게만 설정했을 경우 읽기 전용이 되어 staff1.name = "핑크퐁"; 이런식으로 초기화를 하려고 접근하는 경우 오류가 발생하게 된다.(불러오는 것만 가능하도록 설정이 되어 있으므로)
