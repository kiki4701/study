Linq 는 데이터 진리 기능이 있는데 원하는 데이터를 찾아낼 때 사용하게 된다.

예를 들어 10살 이상의 데이터를 출력하고 싶을 경우

if(a.age >= 10)

{

    Console.WriteLine(데이터);

}

라고 데이터를 뽑아냈었는데 Linq를 사용하면

var linq = from a in ch where a.age >= 10 select a;

이런식으로 sql문과 비슷한 형식으로 입력을 하게 되면 linq 변수에 해당 조건의 변수들이 다 담기게 된다.

조건이 복잡해질 경우 Linq로 데이터를 뽑아내는것이 코드를 훨씬 더 간결하게 작성할 수 있다.
