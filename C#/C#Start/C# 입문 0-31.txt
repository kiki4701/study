C# 입문 0-31

람다식은 코드를 간결하게 사용하기 위해 사용한다.

Delegate 를 사용할 때 익명메소드를 사용하면

Calc cal;

cal = delegate(int a, int b)

{

    return a + b;

};

이렇게 사용하는 것을 람다식 방식으로 하면

Calc cal2= (int x, int y) => x + y;

이렇게 간단하게 사용할 수 있다.

람다식은 간단하게 '식' 형식으로 되어있으며 인스턴스화 할때 =>를 기준으로 왼쪽은 매개변수, 오른쪽은 '식'이 위치하게 된다.

문 형식 람다식이라는 것도 있는데 얘는 인스턴스화 할때 =>를 기준으로 왼쪽에는 매개변수, 오른쪽에는 식이 아닌 {}로 둘러싸인 코드블럭이 위치하게 된다.

예를들면

DelegateLambda dl = () =>

{

    Console.WriteLine("문 형식의 람다입니다.");

}
