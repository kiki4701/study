C# 입문 0-5

ctrl + k + c 해당 블럭 한번에 주석처리

ctrl + a (전체 선택) -> ctrl + k + f (선택 영역 자동 코드 정렬)

예전에 이클립스 ctrl + shift + i 랑 같은것 같음

var : 처음 할당된 데이터타입 int일경우 그 다음 할당도 int만 가능,

다른 변수에 데이터 타입이 같으면 바로 할당가능 (앞 예시에서는 int)

object : 처음 할당된 데이터 타입을 변경할 수 있음, 예를들면 처음에 변수에 int를 초기화하고 그다음에 그 변수를 string으로 다시 초기화해도 된다.

하지만 object a = 1; 이렇게 int값으로 초기화 해도 int b = a;

이렇게 다시 초기화 해줄순 없다. (object를 초기화 할 때 int값을 넣어서 초기화 되긴하지만 초기화가 끝나고 난 후에는 a는 int 자료형이 아닌 object 라는 다른 자료형으로 판단되기 때문)

dynamic : 처음 할당된 데이터타입을 변경할 수 있다.

다른 변수에 데이터 타입이 같으면 바로 할당 가능하다.(유동성이 제일 좋음)

하지만 dynamic k = "hello";

int i = k; 이렇게 해도 오류표시는 나지 않지만 막상 실행 할 때 오류가 나게 된다(int 에 string 을 할당하려고 하니 당연한것, 오류 표시만 안난다)
